# FrontFaraday
depot de mon projet faraday
Faraday est une plateforme open-source dédiée à la gestion centralisée des vulnérabilités. Elle regroupe automatiquement les résultats de divers outils de pentest et de scanning pour offrir une vision unifiée de l’état de sécurité d’un système. Grâce à son interface collaborative, Faraday permet aux équipes de cybersécurité de suivre les failles, prioriser les actions, assigner des tâches et documenter tout le processus de remédiation. C’est un outil conçu pour rendre les audits plus organisés, plus rapides et plus efficaces.
